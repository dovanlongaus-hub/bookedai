# BookedAI frame prompts

## Usage
Use each prompt as a separate generation prompt for image creator, storyboard creator, or video scene planning.

## Global style prompt
Premium dark SaaS visual style, cinematic but restrained, cyan and violet accents, subtle glassmorphism, clean typography, calm confidence, modern AI-native business aesthetic, high contrast, minimal clutter, polished product-marketing art direction, 16:9 composition, investment-grade visual quality.

---

## Frame 1 prompt, The problem
Create a 16:9 cinematic keyframe for a premium SaaS promo video. Show a dark gradient background with scattered signals representing search, calls, emails, and enquiries. The scene should feel fragmented and disconnected, with floating UI chips or abstract notification fragments drifting in space. Add one central glassmorphism panel trying to organize the chaos. Keep the composition clean, premium, and minimal. No stock-photo people. No generic robot imagery. Text overlay area should support the line: “Service businesses do not need more tools.”

---

## Frame 2 prompt, One revenue flow
Create a 16:9 premium SaaS keyframe where scattered business signals converge into one clear flow. Use a dark high-end interface look inspired by BookedAI product marketing. Show directional motion or a unifying light path that pulls multiple inputs into one central system. The scene should communicate operational clarity and momentum. Clean typography space for: “They need one revenue flow.” Keep the look strategic, polished, and investment-grade.

---

## Frame 3 prompt, AI conversation capture
Create a 16:9 premium mobile-product keyframe showing an AI booking assistant as the first intelligent touchpoint. The screen should feel like a real high-end SaaS mobile UI, dark mode, cyan and violet highlights, minimal but powerful interface. Highlight the input area or live indicator to show that the AI assistant is actively capturing intent. The scene should feel focused, responsive, and intelligent. Text overlay area for: “AI conversation captures intent.”

---

## Frame 4 prompt, Intent becomes bookable action
Create a 16:9 premium SaaS keyframe showing a booking flow where intent turns into bookable action. The visual focus should be a clean appointment availability interface with time slots, service matching, and one clearly highlighted selection. The style should feel real, product-led, modern, and elegant. Use cyan highlighting on the chosen appointment slot. Preserve strong legibility and avoid clutter. Text overlay area for: “Intent becomes bookable action.”

---

## Frame 5 prompt, Booking confirmed
Create a 16:9 polished SaaS keyframe showing a successful booking confirmation state. The composition should feel stable, reassuring, and outcome-focused. Feature appointment details, confirmation UI, and a subtle success treatment such as a soft glow or confirmation check emphasis. Keep the design minimal and trustworthy, with premium dark styling and restrained accent color use. Text overlay area for: “Booking confirmed.”

---

## Frame 6 prompt, Full workflow on desktop
Create a 16:9 premium desktop product keyframe showing a broader workflow from search to results to booking. The screen should feel like a serious SaaS product, not a generic landing page. Show connected panels or a continuous journey across the interface that suggests one unified system. Use dark mode with cyan and violet accents, clean information hierarchy, and enough whitespace to keep the UI readable. Text overlay area for: “From inquiry to booking flow.”

---

## Frame 7 prompt, Operator control
Create a 16:9 premium control-center keyframe for a SaaS platform. Show a dark operator dashboard with four clear zones or tiles representing CRM sync, payments, tenants, and audit. The frame should feel structured, trustworthy, and high-value, like the backend command center behind an AI-first booking product. Avoid noisy fake data tables and generic dashboard clutter. Use balanced composition, glassmorphism support, and calm premium lighting. Text overlay area for: “CRM sync. Payments. Tenants. Audit.”

---

## Frame 8 prompt, Brand close
Create a 16:9 premium final brand frame for BookedAI. Use a dark hero background with elegant ambient gradients, subtle blurred UI collage in the background, and a centered or near-centered premium logo area. The frame should feel like a polished end card for an investor-grade SaaS promo video. Minimal text, strong contrast, luxurious restraint, and clear brand focus. Text overlay should support: “BookedAI, AI revenue engine for service businesses.”

---

## Optional animation prompt add-on
Use slow cinematic camera motion, gentle push-ins, subtle parallax, restrained glow effects, smooth premium transitions, and product-legible framing. Avoid flashy meme editing, aggressive zooms, generic futuristic robots, or noisy motion graphics.

## Optional consistency add-on
Keep all 8 frames visually consistent as one campaign: same color system, same typography mood, same premium dark SaaS lighting, same cinematic restraint, same product-first composition language.
