# Prompt patterns

## Image creative

Use this shape:
- objective
- subject
- composition
- style
- lighting
- color palette
- brand cues
- text treatment
- aspect ratio
- avoid list

Template:

Create a [aspect ratio] visual for [objective]. Show [subject]. Composition: [composition]. Style: [style]. Lighting: [lighting]. Palette: [palette]. Brand cues: [brand cues]. Text treatment: [text treatment]. Avoid: [avoid].

## Storyboard frame

Template:

Create a storyboard keyframe for [scene goal]. Show [subject/action]. Composition: [composition]. Environment: [environment]. Style: [style]. Motion implication: [motion]. Text overlay area for: "[text]". Avoid clutter, generic stock-photo look, and unnecessary UI noise.

## Diagram prompt

Template:

Turn this process into a clean [diagram type]. Include [entities]. Show relationships: [relationships]. Optimize for readability over completeness. Use short labels and balanced spacing.

## Chart prompt

Template:

Create a clean [chart type] showing [metric] by [category/time]. Highlight [main takeaway]. Use a premium, minimal style. Clearly label units and call out the strongest insight.
